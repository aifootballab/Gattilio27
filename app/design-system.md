# Design System - Futuristic Dark Theme

## Colori Neon
- **Primary Blue**: `#00D4FF` (cyan brillante)
- **Primary Purple**: `#A855F7` (viola)
- **Primary Pink**: `#EC4899` (rosa)
- **Primary Orange**: `#FF6B35` (arancione)
- **Background Dark**: `#0A0E27` (blu scuro)
- **Background Darker**: `#050815` (quasi nero)

## Effetti Glow
- `box-shadow: 0 0 20px rgba(0, 212, 255, 0.3)` (blu)
- `box-shadow: 0 0 20px rgba(168, 85, 247, 0.3)` (viola)
- `text-shadow: 0 0 10px rgba(0, 212, 255, 0.8)` (testo glow)

## Background Personalizzabile
- Classe `.custom-background` per inserire il tuo sfondo
- Supporta: `background-image`, `background-color`, gradienti
- Layer separato per effetti overlay

## Componenti
- `.neon-card`: Card con bordo glow
- `.neon-panel`: Pannello con effetti
- `.neon-button`: Pulsanti con glow
- `.neon-text`: Testo con glow
- `.progress-glow`: Barre progresso con glow
